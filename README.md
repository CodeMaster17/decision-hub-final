**Installation**

Install the project dependencies using npm:

- clone the repo
- There are 3 folders
  - mongodb server
  - client
  - postgres backend

# Start the client

`npx vite`

# Start the mongodb server

`npm start`

# Start the postgres backend

`npm start`
